# Saucedemo Automation Using Selenium

This project demonstrates an automated workflow on the Saucedemo website using Selenium and undetected_chromedriver. The automation covers logging in, adding a product to the cart, checking the cart, and logging out.

---

## Features
- Opens the Saucedemo website  
- Logs in using valid credentials  
- Adds a product to the shopping cart  
- Visits the cart page  
- Retrieves the product name  
- Logs out successfully  

---

## How to Run
1. Clone or download this repository  
2. Place your Selenium automation script inside the project folder  
3. Open a terminal inside the folder  
4. Run the script normally using Python  

---

## Notes
- Ensure Chrome is up to date  
- Do not close the browser while the automation is running  
- Use a stable internet connection while testing  
- Keep your script in the correct project directory
